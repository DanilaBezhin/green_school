Политика конфиденциальности персональных данных

Настоящая Политика конфиденциальности (далее — Политика) определяет порядок обработки и защиты информации о физических лицах, использующих сайт green_school (далее — Сайт).

1. Общие положения

1.1. Настоящая Политика разработана в соответствии с Федеральным законом от 27.07.2006 № 152-ФЗ «О персональных данных».

1.2. Владелец сайта (Бежин Данила Владимирович, ИНН 228901828554, зарегистрированный как самозанятый) обязуется обеспечить защиту персональных данных пользователей.

2. Обрабатываемые данные

2.1. При использовании формы обратной связи на сайте собираются следующие данные:

  • Имя пользователя.
  • Номер телефона.

2.2. Данные используются для:

  • Обратной связи с пользователем.
  • Предоставления информации о репетиторских услугах.

3. Цели обработки данных

3.1. Персональные данные обрабатываются для:

  • Установления связи с пользователем.
  • Оказания образовательных услуг.
  • Информирования пользователя о новых предложениях или услугах.

4. Права пользователей

4.1. Пользователь имеет право:

  • Запрашивать уточнение своих персональных данных.
  • Требовать удаления или блокировки данных.
  • Отозвать согласие на обработку данных.

4.2. Для реализации прав пользователь может связаться с владельцем сайта через контактную информацию, указанную внизу страницы.

5. Меры защиты данных

5.1. Владелец сайта принимает технические и организационные меры для защиты персональных данных от несанкционированного доступа, утраты или разглашения.

5.2. Все данные передаются по защищенному протоколу HTTPS.

6. Хранение данных

6.1. Персональные данные хранятся до момента отзыва пользователем согласия на их обработку.

6.2. По истечении срока хранения данные удаляются или анонимизируются.
